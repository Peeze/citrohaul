# Citrohaul

Silly project inspired by Nitrohaul.

## Dependencies

Depends on [Matter.js](https://github.com/liabru/matter-js).

## Controls

Draw objects with mouse. Change object type with keys:
- `1`: Wheel, will spin with `ArrowUp` and `ArrowDown`
- `2`: Circle
- `3`: Plank
- `4`: Box
